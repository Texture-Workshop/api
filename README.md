# Texture Workshop's API & Website
The source code of the [Texture Workshop mod](https://geode-sdk.org/mods/uproxide.textures)'s [API](https://tws.m336.dev/api/v1/tws/getTPs) & its [website](https://tws.m336.dev/)

## Running
1. Clone the repository with `git clone https://github.com/M336G/texture-workshop-api.git`.
2. Install [Node.js](https://nodejs.org/en/download/current) with npm and run `npm i`.
3. Run the project with `npm run start`.
